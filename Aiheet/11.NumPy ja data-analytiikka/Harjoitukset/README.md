# Harjoitukset: NumPy ja data-analytiikka

Tässä kansiossa on harjoitustehtävät NumPy-kirjastosta ja data-analytiikan perusteista.

**HUOM:** Nämä harjoitukset vaativat NumPy-kirjaston. Asenna se ensin: `pip install numpy`

## Harjoituslista

### [Harjoitus 1: NumPy-taulukon perusteet](Harjoitus%201/README.md) ⭐
Luo NumPy-taulukko, tee perusoperaatioita ja tulosta tulokset.

### [Harjoitus 2: Tilastofunktiot](Harjoitus%202/README.md) ⭐⭐
Laske keskiarvo, mediaani, keskihajonta ja muut tilastot datalle.

### [Harjoitus 3: Matriisioperaatiot](Harjoitus%203/README.md) ⭐⭐
Luo matriiseja ja tee perusoperaatioita (yhteenlasku, kertolasku, transpoosi).

### [Harjoitus 4: Datan analysointi](Harjoitus%204/README.md) ⭐⭐⭐
Analysoi opintosuoritusdataa: laske keskiarvot, etsi parhaat/huonoimmat.

### [Harjoitus 5: Lineaarinen regressio](Harjoitus%205/README.md) ⭐⭐⭐⭐
Toteuta yksinkertainen lineaarinen regressio ja tee ennusteita.

---
