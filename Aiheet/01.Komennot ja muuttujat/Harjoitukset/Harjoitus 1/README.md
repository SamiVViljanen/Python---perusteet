# Harjoitus 1: Tervehdys (⭐ Helppo)

## Tavoite
Harjoittele input()- ja print()-funktioiden käyttöä


## Tehtävät

1. Kysy käyttäjältä hänen nimensä
2. Kysy käyttäjältä hänen ikänsä (muista muuntaa int-tyypiksi)
3. Tulosta tervehdys muodossa: "Hei [nimi]! Olet [ikä] vuotta vanha."
