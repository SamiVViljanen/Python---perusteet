# Harjoitus 1: Perus sanakirja ⭐

## Tehtävänanto

Luo sanakirja, jossa on 5 henkilön nimet ja iät.

**Tee seuraavat asiat:**
1. Luo sanakirja nimellä `henkilöt`
2. Lisää 5 avain-arvo-paria (nimi: ikä)
3. Tulosta koko sanakirja
4. Tulosta yhden henkilön ikä käyttämällä nimeä avaimena
5. Lisää yksi uusi henkilö sanakirjaan
6. Tulosta päivitetty sanakirja

## Esimerkkitulostus

```
Kaikki henkilöt:
{'Anna': 25, 'Pekka': 30, 'Liisa': 28, 'Matti': 35, 'Kaisa': 22}

Annan ikä: 25

Lisätään uusi henkilö...

Päivitetyt henkilöt:
{'Anna': 25, 'Pekka': 30, 'Liisa': 28, 'Matti': 35, 'Kaisa': 22, 'Ville': 27}
```

## Vinkkejä

- Luo sanakirja: `henkilöt = {"Anna": 25, "Pekka": 30}`
- Hae arvo: `henkilöt["Anna"]`
- Lisää uusi: `henkilöt["Ville"] = 27`
- Tulosta: `print(henkilöt)`
